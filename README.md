# LiHiSTO: List of Hindi STOpwords
A package with a comprehensive list of 820 Hindi Stopwords.

## Installation

```bash
pip install LiHiSTO==0.0.1
```

## Usage

```bash
>>> from stopwords_hindi import hindi_sw
>>> sw = hindi_sw.get_hindi_sw()
>>> len(sw)
820

```
